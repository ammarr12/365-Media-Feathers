# 📊 Compensation & Assistance Dashboard (Power BI)

## 📌 Project Overview
This project is a Power BI dashboard created to analyze and visualize compensation and assistance data. The goal was to transform raw data into meaningful insights that help understand payment distribution, case status, and regional trends.

---

## 🎯 Objectives
- Provide a clear overview of total and pending cases  
- Analyze total payments (exceeding $1B+)  
- Break down payment statuses (Disbursed, Approved, Pending)  
- Visualize city-wise distribution of payments  
- Enable detailed case-level insights  

---

## 📈 Dashboard Features

### 🔹 Key Metrics
- Total Cases  
- Pending Cases  
- Total Payments  

### 🔹 Visual Insights
- Payment Status Breakdown (Disbursed, Approved, Pending)  
- City-wise Payment Distribution  
- Summary KPIs for quick understanding  

### 🔹 Detailed Analysis
- Interactive table with individual case data  
- Filters and slicers for better exploration  

---

## 🛠 Tools & Technologies
- **Power BI** – Dashboard creation and visualization  
- **Power Query** – Data cleaning and transformation  
- **DAX (basic)** – Calculations and measures  

---

## 💡 Key Learnings
- Data cleaning and transformation using Power Query  
- Structuring dashboards for clarity and usability  
- Importance of visual hierarchy and design choices  
- Converting raw datasets into actionable insights  

---

## 🔍 Key Insight
Even a simple dataset can reveal powerful insights when it is well-structured and visualized effectively.
